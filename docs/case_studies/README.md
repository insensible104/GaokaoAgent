# Retrospective Case Studies

This directory is reserved for post-season PathFinder case studies.

Case studies here are not family-facing advice. Each case must document:

- the opportunity claim;
- official source evidence;
- historical rank or major-line anchor;
- counterevidence;
- family hard-stop questions;
- counselor decision;
- blocked gates;
- retrospective outcome audit once actual admission outcomes are available.

Do not create broad multi-case reports here until one case has been audited end to end.
